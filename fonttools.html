        字体文件太大怎么办？使用Python库fonttools大幅度压缩字体的大小，实现字体几百KB大小

        本文又是一篇水教程，作为一个Python爱好者，准备使用fonttools库来对字体进行压缩。具体的效果拭目以待，我也不敢保障什么。使用的字体是一个新的字体，并非小陶之前推荐使用的鸿蒙字体『本站更换了鸿蒙Sans字体_博客字体_华为鸿蒙字体下载』，而是标小智龙珠体，挺小众的一款字体。

        你还在为字体太大而自己不大而苦恼吗？还在为访客说你不行而忧心吗？安装Python，几百kb不是梦。本文也是解答一下之前有站长朋友问我如何把鸿蒙字体压缩到只有几百kb的，其实原理就是把常见字导入（常见的汉字大约数3600字左右），以及常见的中文名符号、26个字母和10个数字。其他一些不常见的字则不会被应用字体，进而实现字体文件的瘦身。此外，转换为woff2格式也是压缩字体文件的方法之一！

        标小智龙珠体字体下载

        在2023年，标小智再次携手猫啃网用心打造了一款卡通字体【标小智龙珠体】，允许任何个人和企业免费使用，包括商用用途，字体风格动感活泼俏皮可爱，非常适合用在卡通类logo设计中。基于日本著名字体设计公司Fontworks免费开源的摇滚乐 One(日文名：ロックンロール)字体进行简体字补充的一款字体。

        安装fonttools库

        在此之前，你需要安装一下python编译器，去Python官网安装一下就行了，这里就不再介绍了！网上的介绍也是铺天盖地，实在不行就在B站搜个教程就行了，非常多。

        安装完Python就可以安装我们的主角fonttools了：

        pip install fonttools

        如果你是Python新手，可能会出现'pip' 不是内部或外部命令，也不是可运行的程序，这是因为没有添加环境变量：

        （1）打开 此电脑 / 我的电脑 / 资源管理器，点击左上角的 属性 ，点击 高级系统设置

        （2）在 系统属性 界面中，选择下方的 环境变量

        （3）看 ...的用户变量 ，点击变量 PATH（或 path ），点击 编辑

        （4）在 编辑环境变量 界面，点击 浏览 ，点击 pip 所在的文件夹（pip.exe在python的安装目录里的Scripts里面），然后一定要全都点击 确定（编辑环境变量-->环境变量-->系统属性，要是没有都点击确定的话，会导致修改失败）
